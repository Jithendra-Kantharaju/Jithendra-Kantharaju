## 🔥 Featured Projects

### 🚀 [GitOps CI/CD Pipeline on AWS EKS](https://github.com/Jithendra-Kantharaju/AWS-GitOps-Automation-Actions)
A production-grade dual-repository GitOps architecture separating Infrastructure as Code (Terraform) from application CI/CD. Fully automates EKS cluster provisioning and deployments via GitHub Actions, with quality gates from Maven, Checkstyle, and SonarCloud. Companion IaC repo: [AWS-GitOps-Automation-IaC](https://github.com/Jithendra-Kantharaju/AWS-GitOps-Automation-IaC)

`Terraform` `GitHub Actions` `Docker` `Amazon ECR` `Helm` `SonarCloud` `EKS`

### 🔐 [DevSecOps Pipeline Implementation](https://github.com/Jithendra-Kantharaju/devsecops-pipeline-tictactoe)
End-to-end DevSecOps pipeline featuring unit testing, static code analysis, Docker image building, Trivy container scanning, and automated deployment via Argo CD to Kubernetes — with GitOps-based continuous delivery and zero manual intervention.

`GitHub Actions` `Argo CD` `Docker` `Trivy` `Kubernetes` `GitOps`

### 🎬 [Multi-Tier Cloud Application on AWS](https://github.com/Jithendra-Kantharaju/Movie-Streaming-App-DevOps)
Multi-container application deployed on AWS EKS with a Jenkins CI/CD pipeline that builds, tests, and deploys Dockerized microservices. Integrates S3 for object storage, EC2 for build agents, and RDS (MySQL) for persistent data.

`Jenkins` `Docker` `AWS EKS` `S3` `RDS` `Nginx` `Node.js`
